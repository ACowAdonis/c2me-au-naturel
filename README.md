## About This Fork

This is a personal fork of [C2ME-fabric](https://github.com/RelativityMC/C2ME-fabric) by [ishland](https://github.com/ishland), maintained for experimental modifications and testing.

### Original Project

C^2M-Engine (Concurrent Chunk Management Engine) is a Fabric mod designed to improve the performance of chunk generation, I/O, and loading by taking advantage of multiple CPU cores in parallel.

**Original repository:** https://github.com/RelativityMC/C2ME-fabric
**Original author:** ishland
**Original license:** MIT

## Fork Modifications

This fork includes experimental changes for:
- Spawn blacklist functionality for entity management during worldgen
- Performance optimizations under investigation
- Bug fixes and testing modifications

## Minecraft Version

This fork is based on Minecraft 1.20.1.

## Building

JDK 17+ is required to build and use C2ME.

```shell
./gradlew clean build
```
